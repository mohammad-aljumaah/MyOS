# MyOS

A hobby operating system built from scratch in x86 assembly.

## Author

**Mohammad Aljumaah**

[Website](https://mohammad-aljumaah.me)

## Build & Run

Note: Requires `nasm`, `make`, `qemu-system-i386`.

### Build

```bash
make
```

### Run

```bash
make run
```

### Clean

```bash
make clean
```
> Removes build/
